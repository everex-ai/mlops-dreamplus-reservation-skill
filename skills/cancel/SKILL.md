# 드림플러스 예약 취소

내 예약을 특정해 취소한다. **쓰기 작업이므로 실행 전 반드시 사용자 확인.**

> 경로: 이 스킬 base 디렉토리의 **두 단계 상위가 플러그인 루트**(`REPO`). 스크립트=`REPO/bin/`,
> 공통 절차=`REPO/docs/skill-runtime.md`.

## 절차

1. **프리플라이트** — 로그인 확인(미로그인 안내 후 중단). `myId` 확보.
2. **데이터 가져오기** — `REPO/docs/skill-runtime.md`의 **"내 예약만"** in-page fetch로
   `memberId === myId`인 예약(제목 포함)을 받는다. `RELOAD`면 새로고침 후 재시도.
3. **대상 특정**:
   `echo '<json>' | node REPO/bin/cancel.mjs [<회의실> [시작 HH:mm]] [--id <N>] --date <YYYY.MM.DD> --myid <myId>`
   - 대상 1개면 마지막 줄 `@@ACTION@@ cancel {"id":N}`.
   - 여러 개면 목록을 보여주므로 사용자에게 물어 `--id`로 좁힌다.
4. **사용자 확인** — 취소 대상(회의실·시간·제목)을 보여주고 **취소할지 확인**받는다.
5. **취소 실행(브라우저)** — 확인되면 `REPO/docs/skill-runtime.md`의 "예약 취소" in-page fetch를
   해당 id로 실행. `code==="200"`이면 완료 안내. (같은 날 취소 시 포인트 환불, 시작 30분 전까지만.)
